# 🎨 챌린지 출석 봇 (CheckItOut)

디스코드 공부 챌린지(크로키, 스터디, 코딩 등)의 출석·지각·결석을 자동으로 확인하고 주간 벌금까지 정산해주는 봇입니다.

---

## 📁 파일 구조

```
croquis_bot/
├── bot.py          # 봇 메인 코드
├── config.py       # 초기 기본값 설정
├── settings.py     # 런타임 설정 관리
├── settings.json   # 디스코드에서 변경한 설정 저장 (자동 생성)
├── .env            # 봇 토큰 (직접 생성 필요)
├── .env.example    # .env 샘플
└── requirements.txt
```

---

## 🚀 로컬 실행 방법

### 1. 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 봇 토큰 설정

```bash
cp .env.example .env
# .env 파일을 열어서 DISCORD_TOKEN= 뒤에 봇 토큰 입력
```

Discord Developer Portal → Bot 탭 → Token → Copy

### 3. Developer Portal 설정

Bot 탭 → Privileged Gateway Intents에서 아래 두 항목 활성화:

- ✅ `MESSAGE CONTENT INTENT`
- ✅ `SERVER MEMBERS INTENT`

### 4. 봇 서버 초대

OAuth2 → URL Generator에서 체크 후 초대 링크 생성:

- Scopes: `bot`, `applications.commands`
- Bot Permissions: `채널 보기`, `메시지 보내기`, `링크 임베드`, `파일 첨부`, `메시지 기록 보기`

### 5. 실행

```bash
python bot.py
```

## ⚙️ 초기 기본값 (`config.py`)

디스코드 내 `/설정변경~` 커맨드로 변경한 값은 `settings.json`에 저장됩니다.
`config.py`는 봇을 처음 실행할 때의 기본값으로만 사용됩니다.

| 항목           | 변수명                                    | 기본값                |
| -------------- | ----------------------------------------- | --------------------- |
| 하루 기준 시각 | `DAY_START_HOUR`                          | `6` (오전 6시)        |
| 자동 발표 시각 | `AUTO_REPORT_HOUR` / `AUTO_REPORT_MINUTE` | `5:50`                |
| 참여일         | `CHALLENGE_DAYS`                          | `[0,1,2,3,4]` (월~금) |
| 채널 접두사    | `CHANNEL_PREFIX`                          | `"크로키-"`           |
| 출석 채널      | `ATTENDANCE_CHANNEL_NAME`                 | `"출석체크"`          |
| 타임존         | `TIMEZONE`                                | `"Asia/Seoul"`        |

| `/설정확인`                 | 현재 봇 설정 전체 확인              |

| 커맨드              | 설명                                               |
| ------------------- | -------------------------------------------------- |
| `/설정변경출석채널` | 출석 결과 발표 채널 변경                           |

## 🤖 자동 실행

| 시각                               | 동작                            |
| ---------------------------------- | ------------------------------- |
| 매일 설정된 발표 시각 (기본 05:50) | 출석 현황 자동 발표 → 출석 채널 |
| 매일 00:00                         | 미참여자 멘션 알림 → 출석 채널  |
| 매주 마지막 챌린지 요일 발표 시각  | 주간 정산 자동 발표 → 정산 채널 |

## 📋 출석·지각·결석 판정 기준

| 상황                             | 판정                        |
| -------------------------------- | --------------------------- |
| 당일 이미지 1장 이상 업로드      | ✅ 정상                     |
| 당일 0장, 다음날 2장 이상 업로드 | ⏰ 전날 지각 + ✅ 당일 정상 |
| 당일 0장, 다음날 1장 업로드      | ❌ 전날 결석 + ✅ 당일 정상 |
| 끝까지 0장                       | ❌ 결석                     |

업로드 기준 시각 예시 (`DAY_START_HOUR=6`):

> 6월 10일(월) = 6월 10일 06:00 ~ 6월 11일 05:59

## 🗂️ 채널 이름 규칙

`/설정변경접두사`로 설정한 접두사로 시작하는 채널을 자동으로 감지합니다.

```
접두사: 크로키-
크로키-나나  →  참여자: 나나
크로키-미미  →  참여자: 미미

접두사: 스터디-
스터디-나나  →  참여자: 나나
```

참여자 멘션 연결은 `/참여자등록` 으로 별도 설정해야 합니다.
